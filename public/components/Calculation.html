HRDD Risk Assessment Tool

Mathematical Formulation & Implementation Guide

Focus-Based Risk Allocation with Country-Specific Coverage Distribution
December 2024

1) Overview

This document provides the complete mathematical formulation for a coverage-based Human Rights Due Diligence (HRDD) risk assessment system. The methodology incorporates country-specific coverage distribution, focus-based targeting, and real-time effectiveness tracking to optimize risk management strategies across multinational supply chains.

2) Portfolio Risk Foundation

1Volume-Weighted Baseline Risk

B = (Σᵢ wᵢ · bᵢ) / Σᵢ wᵢ

Where:

  • wᵢ = volume weight for country i
  • bᵢ = baseline risk score for country i (0-100)
  • B = portfolio baseline risk

2Risk Concentration Factor

C = (Σᵢ (wᵢ/W) · bᵢ²) / B²

Properties:

  • C ≥ 1 always (mathematical constraint)
  • C = 1 when all countries have identical risk
  • C > 1 indicates risk concentration in specific countries
  • Higher C values amplify focus effectiveness

3) Focus-Based Coverage Distribution

Coverage is distributed individually to each country based on their risk profile, focus settings, and resource conservation constraints.

1Focus Exponent Calculation

φ = { 1 + f · (φₘₐₓ - 1) · (f/0.5) if f ≤ 0.5 1 + (φₘₐₓ - 1) · [0.5 + (f-0.5)^1.8 · 0.5] if f > 0.5 } Where: φₘₐₓ = 3.5

Characteristics:

  • Linear growth up to 50% focus threshold
  • Accelerated exponential growth beyond 50% (exponent 1.8)
  • Maximum exponent of 3.5 for aggressive targeting

2Biased Risk Ratio with Compression

rᵢ' = max(rₘᵢₙ, min(rₘₐₓ, (bᵢ/B)^φ)) If f > 0.6 AND bᵢ/B < 0.8: rᵢ' = rᵢ' · [0.3 + 0.7 · (bᵢ/B)²]

Parameters:

Parameter Value Impact
rₘᵢₙ 0.05 Strong low-risk penalties
rₘₐₓ 5.0 High high-risk benefits
Compression Quadratic Additional low-risk reduction

3Country-Specific Coverage Allocation

cᵢⱼ = cⱼ · [(1-f) + f · rᵢ'] · ηᵢ If f > 0.6 AND bᵢ ≥ 60: cᵢⱼ = cᵢⱼ · [1 + (f-0.6) · 0.8]

Where:

  • cᵢⱼ = coverage for country i, tool j
  • cⱼ = base coverage for tool j
  • ηᵢ = resource conservation factor
  • Additional 0.8x boost for very high-risk countries at high focus

Resource Conservation

The system maintains total resource usage close to original levels while allowing up to 30% expansion for high-focus scenarios to accommodate targeting benefits.

4) Country-Specific Transparency Calculation

1Per-Country Category Transparency

For each country i and category k:

Tᵢₖ = 1 - ∏ⱼ∈Jₖ (1 - (cᵢⱼ/100) · (eⱼᵇᵃˢᵉ + eⱼᵘˢᵉʳ)/2)

Tool Categories:

Category Tools Base Effectiveness Category Weight (αₖ)
Worker Voice Continuous (j=0), Surveys (j=1) 90%, 45% 1.0
Audit Unannounced (j=2), Announced (j=3) 25%, 15% 0.85
Passive Self-Report (j=4), Desk-Based (j=5) 12%, 5% 0.70

2Combined Country Transparency

Tᵢ = min(1 - ∏ₖ₌₁³ (1 - Tᵢₖ · αₖ), 0.90)

3Portfolio-Wide Transparency

T = (Σᵢ wᵢ · Tᵢ) / Σᵢ wᵢ

This represents the volume-weighted average transparency across all countries in the portfolio, accounting for their individual coverage allocations.

5) Focus Multiplier System

1Portfolio Focus Multiplier

Fₘ = (1 - f · γ) + f · γ · C

Where: γ = 2.75 (focus concentration weight)

2Country-Specific Focus Multipliers

For each country i: If f > 0.6 AND bᵢ ≥ 70: Fᵢ = Fₘ · [1 + (f-0.6) · 0.5] Else: Fᵢ = (1 - f · γ) + f · γ · rᵢ'

Ultra-High-Risk Bonus: Countries with baseline risk ≥70 and focus >60% receive additional multiplier benefits up to 20% extra effectiveness.

6) Response Effectiveness

R = (Σⱼ rⱼ · ρⱼ) / Σⱼ rⱼ

Where:

  • rⱼ = response strategy weight for method j
  • ρⱼ = response effectiveness rate for method j (0-100%)

7) Final Managed Risk Calculation

1Individual Country Managed Risk

mᵢ = max(0, bᵢ · [1 - Tᵢ · R · Fᵢ])

2Portfolio Managed Risk

M = (Σᵢ wᵢ · mᵢ) / Σᵢ wᵢ

8) Focus Effectiveness Metrics

The system provides comprehensive metrics to evaluate how effectively the focus strategy is targeting high-risk countries.

Risk Tier Classification

  • High Risk: bᵢ ≥ 60
  • Medium Risk: 40 ≤ bᵢ < 60
  • Low Risk: bᵢ < 40

Effectiveness Measures

For each risk tier: ē_tier = (Σᵢ∈tier (bᵢ - mᵢ)/bᵢ · 100) / |tier| Differential Benefit = ē_high - ē_low Focus Effectiveness = min(100, (Differential Benefit)/(f · 50) · 100)

Performance Indicators:

  • Focus Effectiveness ≥70%: Excellent targeting
  • Focus Effectiveness 40-70%: Moderate targeting
  • Focus Effectiveness <40%: Poor targeting, strategy adjustment needed

10) Worked Example

Portfolio Setup

Three countries with focus level f = 0.8:

  • Country A: wₐ = 0.5, bₐ = 80 High Risk
  • Country B: wᵦ = 0.3, bᵦ = 50 Medium Risk
  • Country C: wᵧ = 0.2, bᵧ = 20 Low Risk

Step 1: Portfolio Metrics

B = (0.5×80 + 0.3×50 + 0.2×20) / (0.5+0.3+0.2) = 59 C = [(0.5×80² + 0.3×50² + 0.2×20²) / (0.5+0.3+0.2)] / 59² = 1.42

Step 2: Focus Calculations

φ = 1 + (3.5-1) × [0.5 + (0.8-0.5)^1.8 × 0.5] = 2.65 Country A: rₐ' = min(5, (80/59)^2.65) = 2.84 Country B: rᵦ' = (50/59)^2.65 = 0.53 Country C: rᵧ' = max(0.05, (20/59)^2.65) = 0.05 (compression applied)

Step 3: Coverage Allocation Example (Unannounced Audits, base 25%)

Country A: cₐ = 25 × [(1-0.8) + 0.8×2.84] × 1.32 = 82.9% (High-risk bonus: 82.9 × 1.16 = 96.2%) Country B: cᵦ = 25 × [(1-0.8) + 0.8×0.53] = 15.6% Country C: cᵧ = 25 × [(1-0.8) + 0.8×0.05] = 6.0%

Step 4: Expected Outcomes

  • Country A: Receives 96% audit coverage (high targeting)
  • Country B: Receives 16% audit coverage (moderate reduction)
  • Country C: Receives 6% audit coverage (minimal allocation)
  • Differential Benefit: High-risk countries receive 16x more coverage than low-risk
  • Resource Conservation: Total usage ≈ 130% of baseline (within 30% limit)

11) Key Behavioral Properties

Monotonic Coverage Response

Increasing coverage for any tool always increases transparency T, ensuring predictable and logical system behavior.

Realistic Bounds

  • Maximum transparency: 90% (some violations always remain hidden)
  • Minimum managed risk: 0 (cannot be negative)
  • Country coverage: 0-100% (practical implementation limits)

Focus Effectiveness Validation

  • Real-time measurement prevents ineffective focus strategies
  • Differential benefit tracking ensures high-risk countries receive meaningful advantages
  • Performance scoring guides strategy optimization

Resource Conservation with Flexibility

  • Base resource usage maintained across portfolio
  • Up to 30% expansion allowed for high-focus scenarios
  • Proportional scaling preserves relative allocations
